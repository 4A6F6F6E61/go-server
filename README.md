# Go Backend Server for my Flutter App (WIP)

## File Server

```yml
url: localhost:3000/
method: GET
```

## Api Endpoints

### Get Comics

```yml
url:          http://localhost:3000/api/comics
method:       GET
Content-Type: JSON
```

### Get Videos

```yml
url:          http://localhost:3000/api/videos
method:       GET
Content-Type: JSON
```

### Get Pictures

```yml
url:          http://localhost:3000/api/pictures
method:       GET
Content-Type: JSON
```
